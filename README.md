# Harry Borishade Portfolio

This is the simplified, recruiter-friendly version of the portfolio.

Upload these files directly to the root of the `harryborishade.github.io` repository:

- `index.html`
- `style.css`
- `script.js`
- `Harry_Borishade.pdf`

Do not place them inside another folder.

After uploading, wait a minute and refresh the live site with `Ctrl + F5`.
